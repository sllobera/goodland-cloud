  • Dịch vụ quản trị (khởi tạo, FW, vá lỗi, sao lưu, giám sát 24/7) — IDS ghi miễn phí trong thời gian sử dụng.

5. Bản quyền Microsoft SQL Server Enterprise
Báo giá IDS có đính kèm SQL Enterprise trên cả hai máy. Goodland Cloud mặc định chào hạ tầng + Windows Server; SQL Enterprise có thể: (i) BYOL của Đại học Dầu Khí (giấy phép Volume/Education nếu đủ điều kiện), hoặc (ii) SPLA / CSP theo usage — cần bảng kê core/socket để báo giá riêng (thường là dòng tiền đáng kể so với chỉ Windows + VM). Vui lòng xác nhận mô hình license để chốt giá cuối.

6. Điều khoản gợi ý

  • Thanh toán: tương tự IDS (30% / 70%) hoặc theo thỏa thuận.
  • Thời gian triển khai: 1–2 tuần sau PO và đợt thanh toán đầu.
  • Hiệu lực báo giá: 30 ngày.
  • SLA uptime & hỗ trợ: theo phụ lục Business / Enterprise trong tài liệu Goodland Cloud (gửi kèm khi chốt hợp đồng).
